# QA-Sync Mobile - Actividad 8 Etapa 3

App móvil desarrollada con **Flet** y **Firebase Cloud Firestore** para el reporte de incidencias técnicas en tiempo real.

## Pruebas Realizadas
- **Pruebas Estáticas:** Revisión de lógica de negocio y seguridad en el manejo de credenciales de Firebase.
- **Pruebas Dinámicas:** Verificación de sincronización bidireccional entre la App y la nube.
- **Seguridad:** Implementación de principios de "Privacidad por Diseño" (exclusión de llaves privadas en el repo).

## Cómo ejecutar
1. Instalar dependencias: `pip install -r requirements.txt`
2. Ejecutar: `python main.py`
